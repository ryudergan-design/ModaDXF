# ModaDXF - Listing Adobe

## Dados principais

- Nome: ModaDXF
- Versao inicial: 0.1.4
- Preco: gratis
- Categoria: Tools & Automation
- Host: Adobe Illustrator
- Plataforma: Windows x64
- Autor/publicador: Jeferson Conceicao

## Descricao curta

Importador DXF para Illustrator focado em moldes de moda, preservando agrupamentos principais de arquivos Modaris.

## Descricao longa

ModaDXF ajuda o Adobe Illustrator a abrir moldes em DXF com estrutura mais organizada. O plugin le diretamente o DXF, reconstrui os grupos principais derivados de `BLOCK`/`INSERT`, preserva camadas quando possivel e gera logs locais para diagnostico.

O plugin foi criado para fluxo de moda, especialmente arquivos DXF exportados por sistemas de modelagem como Modaris.

## Onde encontrar

No Illustrator, abrir/importar um arquivo `.DXF` e escolher `ModaDXF Modaris DXF` quando o Illustrator perguntar o formato.

## Permissoes e privacidade

- Nao usa internet.
- Nao envia telemetria.
- Nao coleta dados pessoais.
- Grava apenas logs locais em `%LOCALAPPDATA%\ModaDXF\logs`.

## Validacao recomendada

Validar com arquivos DXF reais de modelagem de moda contendo multiplos `BLOCK`/`INSERT`, confirmando que cada item principal vira um grupo no Illustrator.
